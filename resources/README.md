# Resources

This folder is reserved for shared project support material that does not belong directly inside:

- `android-app/`
- `browser-extension/`
- `backend/`
- `website/`

Current guidance:

- keep release-ready product assets in the platform-specific folders that actually consume them
- keep one-off planning or scratch files out of this folder
- if a new shared asset is added here, document where it is used
