    How it works

    1. Context Menu
    Right-click any link or highlighted phone number on a webpage and select "Send to...".

    2. Quick Send
    Use the text box on the main screen to quickly send a link or phone number to a selected active device.

    3. Seamless Magic
    The SupaPhone Android app instantly receives the data and performs the action (opening the link or starting a call).
